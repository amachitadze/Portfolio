
# ­Ъџђ рЃърЃарЃћрЃЏрЃўрЃБрЃЏ рЃърЃЮрЃарЃбрЃцрЃЮрЃџрЃўрЃЮ & рЃЉрЃарЃћрЃюрЃЊрЃЉрЃБрЃЦрЃў

рЃћрЃА рЃљрЃарЃўрЃА рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃърЃЮрЃарЃбрЃцрЃЮрЃџрЃўрЃЮ рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃБрЃљрЃ«рЃџрЃћрЃАрЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃўрЃЌ (React, Tailwind CSS, Supabase).

## ­ЪЏа Supabase-рЃўрЃА рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃљ (SQL)
Supabase-рЃерЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃњрЃљрЃАрЃљрЃЏрЃљрЃарЃЌрЃљрЃЊ, рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ **SQL Editor**-рЃерЃў рЃЊрЃљ рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў SQL рЃЎрЃЮрЃЊрЃў.

## ­ЪћЉ Environment Variables (Vercel-рЃќрЃћ рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃљ)
рЃўрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃЏ рЃАрЃгрЃЮрЃарЃљрЃЊ рЃўрЃЏрЃБрЃерЃљрЃЮрЃА Vercel-рЃќрЃћ, **Settings -> Environment Variables**-рЃерЃў рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў:

1.  `SUPABASE_URL` - рЃЌрЃЦрЃЋрЃћрЃюрЃў Supabase рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА URL.
2.  `SUPABASE_ANON_KEY` - рЃЌрЃЦрЃЋрЃћрЃюрЃў Supabase рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА Anon Key.
3.  `ADMIN_PASSWORD` - рЃљрЃЊрЃЏрЃўрЃю рЃърЃљрЃюрЃћрЃџрЃўрЃА рЃърЃљрЃарЃЮрЃџрЃў.
4.  `GALLERY_PASSWORD` - рЃњрЃљрЃџрЃћрЃарЃћрЃўрЃА рЃърЃљрЃарЃЮрЃџрЃў.
5.  `IMGBB_API_KEY` - ImgBB-рЃўрЃА API рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃў рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўрЃА рЃљрЃАрЃљрЃбрЃЋрЃўрЃарЃЌрЃљрЃЊ.

**рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў:** рЃЌрЃБ рЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ Vite build-рЃА, рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃА рЃЊрЃљрЃБрЃЏрЃљрЃбрЃћрЃЌ рЃърЃарЃћрЃцрЃўрЃЦрЃАрЃў `VITE_` (рЃЏрЃљрЃњ: `VITE_SUPABASE_URL`). рЃгрЃўрЃюрЃљрЃљрЃдрЃЏрЃЊрЃћрЃњ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃў рЃЏрЃљрЃЌ рЃЋрЃћрЃа рЃЊрЃљрЃўрЃюрЃљрЃ«рЃљрЃЋрЃА.

## ­Ъје рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ
- **рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃћрЃарЃў:** `#32353F`
- **рЃљрЃЦрЃфрЃћрЃюрЃбрЃўрЃА рЃцрЃћрЃарЃў:** `#FFC93C`
- **рЃЉрЃарЃћрЃюрЃЊрЃЉрЃБрЃЦрЃў:** рЃЊрЃљрЃфрЃБрЃџрЃўрЃљ рЃърЃљрЃарЃЮрЃџрЃўрЃЌ. рЃЏрЃљрЃАрЃерЃў рЃърЃарЃЮрЃцрЃўрЃџрЃўрЃА рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЕрЃћрЃюрЃљрЃЊ, рЃњрЃљрЃџрЃћрЃарЃћрЃљрЃерЃў рЃюрЃўрЃЋрЃЌрЃА рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃбрЃћрЃњрЃў: `profile`.

рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃЏрЃљрЃдрЃљрЃџрЃў рЃ«рЃљрЃарЃўрЃАрЃ«рЃўрЃЌ. Рюе
