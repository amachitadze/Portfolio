
# ­Ъџђ рЃърЃарЃћрЃЏрЃўрЃБрЃЏ рЃърЃЮрЃарЃбрЃцрЃЮрЃџрЃўрЃЮ & рЃЉрЃарЃћрЃюрЃЊрЃЉрЃБрЃЦрЃў

рЃћрЃА рЃљрЃарЃўрЃА рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃърЃЮрЃарЃбрЃцрЃЮрЃџрЃўрЃЮ рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃБрЃљрЃ«рЃџрЃћрЃАрЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃўрЃЌ (React, Tailwind CSS, Supabase).

## ­ЪЏа Supabase-рЃўрЃА рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃљ (SQL)
Supabase-рЃерЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃњрЃљрЃАрЃљрЃЏрЃљрЃарЃЌрЃљрЃЊ, рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ **SQL Editor**-рЃерЃў рЃЊрЃљ рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЎрЃЮрЃЊрЃў (рЃћрЃА рЃЎрЃЮрЃЊрЃў рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃљ рЃЊрЃљ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃарЃљрЃЏрЃЊрЃћрЃюрЃ»рЃћрЃарЃЏрЃћ рЃњрЃљрЃБрЃерЃЋрЃљрЃЌ):

```sql
рЃљрЃЦ рЃљрЃдрЃљрЃа рЃЊрЃљрЃгрЃћрЃарЃЮ
```

## ­Ъје рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ
- **рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃћрЃарЃў:** `#32353F`
- **рЃљрЃЦрЃфрЃћрЃюрЃбрЃўрЃА рЃцрЃћрЃарЃў:** `#FFC93C`
- **рЃЉрЃарЃћрЃюрЃЊрЃЉрЃБрЃЦрЃў:** рЃЊрЃљрЃфрЃБрЃџрЃўрЃљ рЃърЃљрЃарЃЮрЃџрЃўрЃЌ. рЃЏрЃљрЃАрЃерЃў рЃърЃарЃЮрЃцрЃўрЃџрЃўрЃА рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЕрЃћрЃюрЃљрЃЊ, рЃњрЃљрЃџрЃћрЃарЃћрЃљрЃерЃў рЃюрЃўрЃЋрЃЌрЃА рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃбрЃћрЃњрЃў: `profile`.

## ­ЪїЇ рЃџрЃЮрЃЎрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ
рЃАрЃљрЃўрЃбрЃў рЃЏрЃ«рЃљрЃарЃА рЃБрЃГрЃћрЃарЃА 3 рЃћрЃюрЃљрЃА: рЃЦрЃљрЃарЃЌрЃБрЃџрЃў (GEO), рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃарЃў (ENG) рЃЊрЃљ рЃћрЃАрЃърЃљрЃюрЃБрЃарЃў (ESP). рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ `/locales` рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃерЃў.

рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃЏрЃљрЃдрЃљрЃџрЃў рЃ«рЃљрЃарЃўрЃАрЃ«рЃўрЃЌ. Рюе
