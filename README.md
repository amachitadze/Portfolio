
# ­Ъџђ рЃърЃарЃћрЃЏрЃўрЃБрЃЏ рЃърЃЮрЃарЃбрЃцрЃЮрЃџрЃўрЃЮ

рЃћрЃА рЃљрЃарЃўрЃА рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃърЃЮрЃарЃбрЃцрЃЮрЃџрЃўрЃЮ рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃБрЃљрЃ«рЃџрЃћрЃАрЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃўрЃЌ (React, Tailwind CSS, Supabase).

## ­Ъје рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ
- **рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃћрЃарЃў:** `#32353F` (рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃбрЃћрЃЦрЃАрЃбрЃўрЃАрЃљ рЃЊрЃљ рЃЉрЃюрЃћрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА).
- **рЃљрЃЦрЃфрЃћрЃюрЃбрЃўрЃА рЃцрЃћрЃарЃў:** `#FFC93C` (рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА).
- **рЃерЃарЃўрЃцрЃбрЃћрЃЉрЃў:** `Google Sans` рЃЊрЃљ `Noto Sans Georgian`.

## Рїе№ИЈ рЃЉрЃћрЃГрЃЊрЃЋрЃўрЃА рЃћрЃцрЃћрЃЦрЃбрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ
рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ рЃЏрЃЊрЃћрЃЉрЃљрЃарЃћрЃЮрЃЉрЃА `components/sections/Hero.tsx`-рЃерЃў:
- `typingSpeed`: рЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃЉрЃА рЃЉрЃћрЃГрЃЊрЃЋрЃўрЃА рЃбрЃћрЃЏрЃърЃА.
- `deletingSpeed`: рЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃЉрЃА рЃгрЃљрЃерЃџрЃўрЃА рЃбрЃћрЃЏрЃърЃА.
- `pauseBeforeDelete`: рЃарЃљрЃЏрЃЊрЃћрЃюрЃў рЃ«рЃљрЃюрЃў рЃњрЃљрЃЕрЃћрЃарЃЊрЃћрЃА рЃбрЃћрЃЦрЃАрЃбрЃў рЃАрЃљрЃюрЃљрЃЏ рЃгрЃљрЃўрЃерЃџрЃћрЃЉрЃљ.

## ­ЪЏа рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ
1. рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў: `npm install`
2. рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃб рЃАрЃћрЃарЃЋрЃћрЃарЃў: `npm run dev`

## ­ЪїЇ рЃџрЃЮрЃЎрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ
рЃАрЃљрЃўрЃбрЃў рЃЏрЃ«рЃљрЃарЃА рЃБрЃГрЃћрЃарЃА 3 рЃћрЃюрЃљрЃА: рЃЦрЃљрЃарЃЌрЃБрЃџрЃў, рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃарЃў рЃЊрЃљ рЃћрЃАрЃърЃљрЃюрЃБрЃарЃў. рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ `/locales` рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃерЃў.

рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃАрЃўрЃДрЃЋрЃљрЃарЃБрЃџрЃўрЃЌ рЃЊрЃљ рЃЏрЃљрЃдрЃљрЃџрЃў рЃ«рЃљрЃарЃўрЃАрЃ«рЃўрЃЌ. Рюе
