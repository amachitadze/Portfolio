
# ­Ъџђ рЃърЃарЃћрЃЏрЃўрЃБрЃЏ рЃърЃЮрЃарЃбрЃцрЃЮрЃџрЃўрЃЮ & рЃЉрЃарЃћрЃюрЃЊрЃЉрЃБрЃЦрЃў

рЃћрЃА рЃљрЃарЃўрЃА рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃърЃЮрЃарЃбрЃцрЃЮрЃџрЃўрЃЮ рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃБрЃљрЃ«рЃџрЃћрЃАрЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃўрЃЌ (React, Tailwind CSS, Supabase).

## ­ЪЏа Supabase-рЃўрЃА рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃљ (SQL)

### рЃЌрЃБ рЃЉрЃљрЃќрЃљ рЃфрЃљрЃарЃўрЃћрЃџрЃўрЃљ (рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ):
рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ Supabase-рЃўрЃА **SQL Editor**-рЃерЃў рЃЊрЃљ рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ:

```sql
ssss
```

### рЃЌрЃБ рЃфрЃ«рЃарЃўрЃџрЃћрЃЉрЃў рЃБрЃЎрЃЋрЃћ рЃњрЃљрЃЦрЃЋрЃЌ (рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ):
рЃЌрЃБ рЃЏрЃўрЃўрЃдрЃћрЃЌ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ `already exists`, рЃћрЃАрЃћ рЃўрЃњрЃў рЃфрЃ«рЃарЃўрЃџрЃў рЃњрЃљрЃЦрЃЋрЃЌ рЃЊрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃљрЃ«рЃљрЃџрЃў рЃАрЃЋрЃћрЃбрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљрЃЌ:

```sql
-- рЃљрЃЏрЃљрЃбрЃћрЃЉрЃА demoUrl рЃАрЃЋрЃћрЃбрЃА рЃљрЃарЃАрЃћрЃЉрЃБрЃџ projects рЃфрЃ«рЃарЃўрЃџрЃерЃў
ccccc
```

## ­ЪћЉ Environment Variables (Vercel-рЃќрЃћ рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃљ)
Vercel-рЃўрЃА **Settings -> Environment Variables**-рЃерЃў рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ:

1.  `SUPABASE_URL` рЃЊрЃљ `VITE_SUPABASE_URL`
2.  `SUPABASE_ANON_KEY` рЃЊрЃљ `VITE_SUPABASE_ANON_KEY`
3.  `VITE_ADMIN_PASSWORD` - рЃљрЃЊрЃЏрЃўрЃю рЃърЃљрЃюрЃћрЃџрЃўрЃА рЃърЃљрЃарЃЮрЃџрЃў.
4.  `VITE_GALLERY_PASSWORD` - рЃњрЃљрЃџрЃћрЃарЃћрЃўрЃА рЃърЃљрЃарЃЮрЃџрЃў.
5.  `VITE_IMGBB_API_KEY` - рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА.
